# FixSpacing: 띄어쓰기 교정기 (Kiwi 기반)

이 프로젝트는 `kiwipiepy`(Kiwi 형태소 분석기)를 사용하여 한국어 텍스트의 잘못된 띄어쓰기를 교정하는 도구입니다. 특히 금융 및 공시 도메인의 전문 용어를 보호하고, 리스트 형태의 잘린 단어들을 복구하는 기능을 포함하고 있습니다.

## 주요 기능

1.  **전문 용어 보호**: '투자신탁', '집합투자기구', '수익증권' 등 금융/법률 관련 용어가 쪼개지지 않도록 사용자 사전을 적용합니다.
2.  **다양한 입력 지원**: 일반 문자열(`str`)뿐만 아니라, 단어가 파편화된 리스트(`list`) 형태의 입력도 처리할 수 있습니다.
3.  **문맥 기반 교정**: 형태소 분석 결과를 바탕으로 조사, 어미, 접미사, 문장 부호 등의 붙임/띔 규칙을 적용합니다.
4.  **사후 보정**: 정규표현식을 사용하여 숫자와 단위(% , 좌), "법 제N조"와 같은 특정 패턴의 띄어쓰기를 추가로 정밀하게 보정합니다.

## 실행 환경 및 설치 방법

### 요구 사항
- Python 3.6 이상
- `kiwipiepy` 라이브러리

### 라이브러리 설치
다음 명령어를 통해 필요한 라이브러리를 설치할 수 있습니다.
```bash
pip install kiwipiepy
```

## 실행 방법

### 기본 실행
터미널에서 `main.py`를 직접 실행하여 포함된 예제들의 결과를 확인할 수 있습니다.
```bash
python main.py
```

### 코드에서 사용 예시
```python
from main import SpacingFixer

fixer = SpacingFixer()

# 문자열 입력 예시
text = "투자자는 수익증권 입고 시점을 사 전에 점검하여 주시기 바랍니다."
print(fixer.fix(text))

# 리스트 입력 예시
text_list = ["수", "익자가", "당해", "환매청구접수의", "취소를", "하지", "아니하였을", "경우"]
print(fixer.fix(text_list))
```
